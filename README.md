# TMPOS Desktop

![Version](https://img.shields.io/badge/version-2.12.8-blue)
![Downloads](https://img.shields.io/github/downloads/tmpos/tmpos-desktop/v2.12.8/total)

Sistema de punto de venta e inventario offline para tiendas, con sincronizacion mediante TMCloud.

## Novedades v2.12.8

- Nuevo modo de tienda general, ocultando las funciones exclusivas de celulares y taller.
- Inventario y operaciones por almacen usando UID, con transferencias de telefonos, electrodomesticos, IMEI y seriales.
- Mejoras en licencias online/offline, configuraciones exportables y envio de OTP por correo.
- Correcciones en POS, apartados, recibidos, busqueda general de IMEI y seleccion de proveedores.
- Reportes financieros corregidos para recargos de tarjeta, gastos por metodo de pago y datos del almacen actual.
- Panel de inicio actualizado con ventas, gastos y ganancias del dia filtrados por almacen.
